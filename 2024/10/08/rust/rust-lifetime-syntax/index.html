2. иҜҜи§Ј

з®ҖеҚ•жқҘи®ІпјҢвјҖдёӘеҸҳйҮҸзҡ„вҪЈе‘Ҫе‘ЁжңҹжҳҜжҢҮвјҖж®өж—¶жңҹпјҢеңЁиҝҷж®өж—¶жңҹеҶ…пјҢиҜҘеҸҳйҮҸжүҖжҢҮеҗ‘зҡ„еҶ…еӯҳең°еқҖдёӯзҡ„ж•°жҚ®жҳҜжңүж•Ҳзҡ„пјҢиҝҷж®өж—¶жңҹжҳҜз”ұзј–иҜ‘еҷЁйқҷжҖҒеҲҶжһҗеҫ—еҮәзҡ„пјҢжңүж•ҲжҖ§з”ұзј–иҜ‘еҷЁдҝқиҜҒгҖӮжҺҘдёӢжқҘжҲ‘е°ҶжҺўи®Ёиҝҷдәӣеёёв»…иҜҜи§Јзҡ„з»ҶиҠӮгҖӮ

1) T еҸӘеҢ…еҗ«жүҖжңүжқғзұ»еһӢ

иҝҷжӣҙеғҸжҳҜеҜ№жіӣеһӢзҡ„иҜҜи§ЈвҪҪвҫ®еҜ№вҪЈе‘Ҫе‘Ёжңҹзҡ„иҜҜи§ЈпјҢдҪҶеңЁ Rust дёӯпјҢжіӣеһӢдёҺвҪЈе‘Ҫе‘Ёжңҹзҡ„е…ізі»жҳҜеҰӮжӯӨзҙ§еҜҶпјҢд»Ҙвҫ„дәҺдёҚеҸҜиғҪеҸӘи®Ёи®әе…¶дёӯвјҖдёӘвҪҪеҝҪи§ҶеҸҰеӨ–вјҖдёӘгҖӮеҪ“жҲ‘еҲҡејҖе§ӢеӯҰд№  Rust ж—¶пјҢжҲ‘зҹҘйҒ“ i32 , &i32 , е’Ң &mut i32 жҳҜдёҚеҗҢзҡ„зұ»еһӢпјҢеҗҢж—¶жҲ‘д№ҹзҹҘжіӣеһӢ T иЎЁзӨәжүҖжңүеҸҜиғҪзұ»еһӢзҡ„йӣҶеҗҲгҖӮз„¶вҪҪпјҢе°Ҫз®ЎиғҪеҲҶеҲ«зҗҶи§ЈиҝҷдёӨдёӘжҰӮеҝөпјҢдҪҶжҲ‘еҚҙжІЎиғҪе°ҶвјҶиҖ…з»“еҗҲиө·жқҘгҖӮеңЁеҪ“ж—¶жҲ‘иҝҷдҪҚ Rust еҲқеӯҰиҖ…зҡ„зңјвҫҘпјҢжіӣеһӢжҳҜиҝҷж ·иҝҗдҪңзҡ„пјҡ

зұ»еһӢT&T&mut T
дҫӢеӯҗi32&i32&mut i32

е…¶дёӯ T еҢ…е…ЁдҪ“жүҖжңүжқғзұ»еһӢпјӣ&T еҢ…жӢ¬е…ЁдҪ“дёҚеҸҜеҸҳеј•вҪӨпјӣ &mut T еҢ…жӢ¬е…ЁдҪ“еҸҜеҸҳеј•вҪӨпјӣT, &T, е’Ң &mut T жҳҜдёҚзӣёдәӨзҡ„жңүйҷҗйӣҶгҖӮз®ҖжҙҒжҳҺдәҶпјҢз¬ҰеҗҲзӣҙи§үпјҢеҚҙе®Ңе…Ёй”ҷиҜҜгҖӮдәӢе®һдёҠжіӣеһӢжҳҜиҝҷж ·иҝҗдҪңзҡ„пјҡ

зұ»еһӢT&T&mut T
дҫӢеӯҗi32, &i32, &mut i32, &&i32, &mut &mut i32&i32, &&i32, &&mut i32&mut i32, &mut &mut i32, &mut &i32

T , &T , е’Ң &mut T йғҪжҳҜвҪҶйҷҗйӣҶпјҢеӣ дёәдҪ еҸҜд»ҘеҖҹвҪӨвјҖдёӘзұ»еһӢвҪҶйҷҗж¬ЎгҖӮT жҳҜ &T е’Ң &mut T зҡ„и¶…йӣҶгҖӮ&T е’Ң &mut T жҳҜдёҚзӣёдәӨзҡ„йӣҶеҗҲ. дёӢвҫҜжңүвјҖдәӣдҫӢвјҰжқҘйӘҢиҜҒиҝҷдәӣжҰӮеҝөпјҡ

дёҠиҝ°д»Јз ҒдёҚиғҪзј–иҜ‘йҖҡиҝҮпјҡ

зј–иҜ‘еҷЁдёҚе…Ғи®ёжҲ‘д»¬дёә &T е’Ң &mut T е®һзҺ° Trait пјҢеӣ дёәиҝҷдёҺжҲ‘д»¬дёә T е®һзҺ°зҡ„ Trait еҸ‘вҪЈдәҶеҶІзӘҒпјҢвҪҪ T е·Із»ҸеҢ…жӢ¬дәҶ &T е’Ң &mut T. еӣ дёә &T е’Ң &mut T жҳҜдёҚзӣёдәӨзҡ„пјҢжүҖд»ҘдёӢвҫҜзҡ„д»Јз ҒеҸҜд»ҘйҖҡиҝҮзј–иҜ‘пјҡ

е…ій”®зӮ№еӣһйЎҫ

  • T жҳҜ &T е’Ң &mut T зҡ„и¶…йӣҶ
  • &T е’Ң &mut T жҳҜдёҚзӣёдәӨзҡ„йӣҶеҗҲ

2) еҰӮжһң T: 'static йӮЈд№Ҳ T зӣҙеҲ°зЁӢеәҸз»“жқҹдёәвҪҢйғҪвјҖе®ҡжҳҜжңүж•Ҳзҡ„

й”ҷиҜҜзҡ„жҺЁи®ә

  • T: 'static еә”иҜҘи§Ҷдёә вҖң T жңүзқҖ 'static вҪЈе‘Ҫе‘ЁжңҹвҖқ
  • &'static T е’Ң T: 'static жҳҜвјҖеӣһдәӢ
  • иӢҘ T: 'static еҲҷ T вјҖе®ҡжҳҜдёҚеҸҜеҸҳзҡ„
  • иӢҘ T: 'static еҲҷ T еҸӘиғҪеңЁзј–иҜ‘жңҹеҲӣе»ә

и®©вјӨеӨҡж•° Rust еҲқеӯҰиҖ…з¬¬вјҖж¬ЎжҺҘи§Ұ 'static вҪЈе‘Ҫе‘ЁжңҹжіЁи§Јзҡ„д»Јз ҒзӨәдҫӢвјӨжҰӮжҳҜиҝҷж ·зҡ„пјҡ

д»–д»¬иў«е‘ҠзҹҘиҜҙ "еӯ—з¬ҰдёІеӯ—вҫҜйҮҸ" жҳҜиў«зЎ¬зј–з ҒеҲ°зј–иҜ‘еҮәжқҘзҡ„вјҶиҝӣеҲ¶вҪӮд»¶еҪ“дёӯеҺ»зҡ„пјҢе№¶еңЁиҝҗвҫҸж—¶иў«еҠ иҪҪеҲ°еҸӘиҜ»еҶ…еӯҳдёӯпјҢжүҖд»Ҙе®ғдёҚеҸҜеҸҳдё”еңЁзЁӢеәҸзҡ„ж•ҙдёӘиҝҗвҫҸжңҹй—ҙйғҪжңүж•ҲпјҢиҝҷд№ҹдҪҝе…¶вҪЈе‘Ҫе‘Ёжңҹдёә 'static. еңЁдәҶи§ЈеҲ° Rust дҪҝвҪӨ static жқҘе®ҡд№үйқҷжҖҒеҸҳйҮҸиҝҷвјҖиҜӯжі•еҗҺпјҢиҝҷвјҖи§ӮзӮ№иҝҳдјҡиў«иҝӣвјҖжӯҘеҠ ејәгҖӮ

е…ідәҺйқҷжҖҒеҸҳйҮҸ

  • е®ғд»¬еҸӘиғҪеңЁзј–иҜ‘жңҹеҲӣе»ә
  • е®ғд»¬еә”еҪ“жҳҜдёҚеҸҜеҸҳзҡ„пјҢдҝ®ж”№йқҷжҖҒеҸҳйҮҸжҳҜ unsafe зҡ„
  • е®ғд»¬еңЁж•ҙдёӘзЁӢеәҸиҝҗвҫҸжңҹй—ҙжңүж•Ҳ

йқҷжҖҒеҸҳйҮҸзҡ„й»ҳи®ӨвҪЈе‘Ҫе‘ЁжңҹеҫҲжңүеҸҜиғҪжҳҜ 'static, еҜ№еҗ§пјҹжүҖд»ҘеҸҜд»ҘеҗҲзҗҶжҺЁжөӢ 'static вҪЈе‘Ҫе‘Ёжңҹд№ҹиҰҒйҒөеҫӘеҗҢж ·зҡ„и§„еҲҷпјҢеҜ№еҗ§пјҹ

зЎ®е®һпјҢдҪҶ жҢҒжңү 'static вҪЈе‘Ҫе‘ЁжңҹжіЁи§Јзҡ„зұ»еһӢе’ҢвјҖдёӘж»Ўвҫң 'static зәҰжқҹ зҡ„зұ»еһӢжҳҜдёҚвјҖж ·зҡ„гҖӮеҗҺиҖ…еҸҜд»ҘдәҺиҝҗвҫҸж—¶иў«еҠЁжҖҒеҲҶй…ҚпјҢиғҪиў«е®үе…Ёвҫғз”ұең°дҝ®ж”№пјҢд№ҹеҸҜд»Ҙиў« drop, иҝҳиғҪеӯҳжҙ»д»»ж„Ҹзҡ„ж—¶в»“гҖӮеҢәеҲҶ &'static T е’Ң T: 'static жҳҜвҫ®еёёйҮҚиҰҒзҡ„вјҖзӮ№гҖӮ&'static T жҳҜвјҖдёӘжҢҮеҗ‘ T зҡ„дёҚеҸҜеҸҳеј•вҪӨпјҢе…¶дёӯ T еҸҜд»Ҙиў«е®үе…Ёең°вҪҶжңҹйҷҗең°жҢҒжңүпјҢз”ҡвҫ„еҸҜд»ҘзӣҙеҲ°зЁӢеәҸз»“жқҹгҖӮиҝҷеҸӘжңүеңЁ T вҫғиә«дёҚеҸҜеҸҳдё”дҝқиҜҒ еңЁеј•вҪӨеҲӣе»әеҗҺ дёҚдјҡиў« move ж—¶жүҚжңүеҸҜиғҪгҖӮT е№¶дёҚйңҖиҰҒеңЁзј–иҜ‘ж—¶еҲӣе»әгҖӮжҲ‘д»¬еҸҜд»Ҙд»ҘеҶ…еӯҳжі„жјҸдёәд»Јд»·пјҢеңЁиҝҗвҫҸж—¶еҠЁжҖҒеҲӣе»әйҡҸжңәж•°жҚ®пјҢе№¶иҝ”еӣһе…¶ 'static еј•вҪӨпјҢвҪҗеҰӮпјҡ

T: 'static жҳҜжҢҮ T еҸҜд»Ҙиў«е®үе…Ёең°вҪҶжңҹйҷҗең°жҢҒжңүпјҢз”ҡвҫ„еҸҜд»ҘзӣҙеҲ°зЁӢеәҸз»“жқҹгҖӮ T: 'static еңЁеҢ…жӢ¬дәҶе…ЁйғЁ &'static T зҡ„еҗҢж—¶пјҢиҝҳеҢ…жӢ¬дәҶе…ЁйғЁжүҖжңүжқғзұ»еһӢпјҢ вҪҗеҰӮ String, Vec зӯүзӯүгҖӮ ж•°жҚ®зҡ„жүҖжңүиҖ…дҝқиҜҒпјҢеҸӘиҰҒвҫғиә«иҝҳжҢҒжңүж•°жҚ®зҡ„жүҖжңүжқғпјҢж•°жҚ®е°ұдёҚдјҡеӨұж•ҲпјҢеӣ жӯӨжүҖжңүиҖ…иғҪеӨҹе®үе…Ёең°вҪҶжңҹйҷҗең°жҢҒжңүе…¶ж•°жҚ®пјҢз”ҡвҫ„еҸҜд»ҘзӣҙеҲ°зЁӢеәҸз»“жқҹгҖӮ T: 'static еә”еҪ“и§Ҷдёә вҖң T ж»Ўвҫң 'static вҪЈе‘Ҫе‘ЁжңҹзәҰжқҹвҖқ вҪҪвҫ® вҖң T жңүзқҖ 'static вҪЈе‘Ҫе‘ЁжңҹвҖқгҖӮ вјҖдёӘзЁӢеәҸеҸҜд»Ҙеё®еҠ©йҳҗиҝ°иҝҷдәӣжҰӮеҝөпјҡ

е…ій”®зӮ№еӣһйЎҫ

  • T: 'static еә”еҪ“и§Ҷдёә вҖң T ж»Ўвҫң 'static вҪЈе‘Ҫе‘ЁжңҹзәҰжқҹвҖқ
  • иӢҘ T: 'static еҲҷ T еҸҜд»ҘжҳҜвјҖдёӘжңү 'static вҪЈе‘Ҫе‘Ёжңҹзҡ„еј•вҪӨзұ»еһӢ жҲ– жҳҜвјҖдёӘжүҖжңүжқғзұ»еһӢ
  • еӣ дёә T: 'static еҢ…жӢ¬дәҶжүҖжңүжқғзұ»еһӢпјҢжүҖд»Ҙ T
    • еҸҜд»ҘеңЁиҝҗвҫҸж—¶еҠЁжҖҒеҲҶй…Қ
    • дёҚйңҖиҰҒеңЁж•ҙдёӘзЁӢеәҸиҝҗвҫҸжңҹй—ҙйғҪжңүж•Ҳ
    • еҸҜд»Ҙе®үе…ЁпјҢвҫғз”ұең°дҝ®ж”№
    • еҸҜд»ҘеңЁиҝҗвҫҸж—¶иў«еҠЁжҖҒзҡ„ drop
    • еҸҜд»ҘжңүдёҚеҗҢв»“еәҰзҡ„вҪЈе‘Ҫе‘Ёжңҹ

3) &'a T е’Ң T: 'a жҳҜвјҖеӣһдәӢ

иҝҷдёӘиҜҜи§ЈжҳҜеүҚвјҖдёӘиҜҜи§Јзҡ„жіӣеҢ–зүҲжң¬гҖӮ&'a T иҰҒжұӮе№¶йҡҗеҗ«дәҶ T: 'a пјҢеӣ дёәеҰӮжһң T жң¬иә«дёҚиғҪеңЁ 'a иҢғеӣҙеҶ…дҝқиҜҒжңүж•ҲпјҢйӮЈд№Ҳе…¶еј•вҪӨд№ҹдёҚиғҪеңЁ вҖҳa иҢғеӣҙеҶ…дҝқиҜҒжңүж•ҲгҖӮдҫӢеҰӮпјҢRust зј–иҜ‘еҷЁдёҚдјҡиҝҗвҫҸжһ„йҖ вјҖдёӘ &'static Ref<'a, T> пјҢеӣ дёәеҰӮжһң Ref еҸӘеңЁ 'a иҢғеӣҙеҶ…жңүж•ҲпјҢжҲ‘д»¬е°ұдёҚиғҪз»ҷе®ғ 'static вҪЈе‘Ҫе‘ЁжңҹгҖӮT: 'a еҢ…жӢ¬дәҶе…ЁдҪ“ &'a T пјҢдҪҶеҸҚд№ӢдёҚжҲҗвҪҙгҖӮ

е…ій”®зӮ№еӣһйЎҫ

  • T: 'a вҪҗ &'a T жӣҙжіӣеҢ–пјҢжӣҙзҒөжҙ»
  • T: 'a жҺҘеҸ—жүҖжңүжқғзұ»еһӢпјҢеҶ…йғЁеҗ«жңүеј•вҪӨзҡ„жүҖжңүжқғзұ»еһӢпјҢе’Ңеј•вҪӨ

д»ҘдёҠиҝҷдәӣеҗ¬иө·жқҘзү№еҲ«еӨҚжқӮпјҢдҪҶжҳҜеҸҜд»Ҙз®ҖеҚ•ең°жҖ»з»“дёәвјҖеҸҘиҜқвҖңвјҖдёӘ trait еҜ№иұЎзҡ„вҪЈе‘Ҫе‘ЁжңҹзәҰжқҹд»ҺдёҠдёӢвҪӮжҺЁеҜјвҪҪеҮәгҖӮвҖқзңӢдёӢвҫҜиҝҷдәӣдҫӢвјҰеҗҺпјҢжҲ‘д»¬дјҡзңӢеҲ°вҪЈе‘Ҫе‘ЁжңҹзәҰжқҹзҡ„жҺЁеҜје…¶е®һеҫҲз¬ҰеҗҲзӣҙи§үпјҢеӣ жӯӨжҲ‘д»¬жІЎеҝ…иҰҒеҺ»и®°еҝҶдёҠвҫҜзҡ„и§„еҲҷпјҡ

вјҖдёӘе®һзҺ°дәҶ trait зҡ„е…·дҪ“зұ»еһӢеҸҜд»Ҙиў«еј•вҪӨпјҢеӣ жӯӨе®ғд»¬д№ҹдјҡжңүвҪЈе‘Ҫе‘ЁжңҹзәҰжқҹпјҢеҗҢж ·е…¶еҜ№еә”зҡ„ trait еҜ№иұЎд№ҹжңүвҪЈе‘Ҫе‘ЁжңҹзәҰжқҹгҖӮдҪ д№ҹеҸҜд»ҘзӣҙжҺҘеҜ№еј•вҪӨе®һзҺ° trait, еј•вҪӨжҳҫз„¶жҳҜжңүвҪЈе‘Ҫе‘ЁжңҹзәҰжқҹзҡ„пјҡ

иҝҷвҫҘзј–иҜ‘еҷЁжҠҘй”ҷпјҡ

еҫҲеҘҪпјҢзј–иҜ‘еҷЁе‘ҠиҜүдәҶжҲ‘д»¬жҖҺж ·дҝ®еӨҚиҝҷдёӘй—®йўҳпјҢи®©жҲ‘д»¬дҝ®еӨҚвјҖдёӢгҖӮ

зҺ°еңЁе®ғзј–иҜ‘йҖҡиҝҮдәҶпјҢдҪҶжҳҜиҝҷдёӨдёӘеҮҪж•°еҜ№вҪҗиө·жқҘзңӢиө·жқҘжҢәеҘҮжҖӘзҡ„пјҢдёәд»Җд№Ҳз¬¬вјҶдёӘеҮҪж•°иҰҒжұӮ T ж»Ўвҫң 'static зәҰжқҹвҪҪз¬¬вјҖдёӘеҮҪж•°дёҚвҪӨе‘ўпјҹиҝҷжҳҜдёӘеҲҒй’»зҡ„й—®йўҳгҖӮдәӢе®һдёҠпјҢйҖҡиҝҮвҪЈе‘Ҫе‘ЁжңҹзңҒз•Ҙи§„еҲҷпјҢRust вҫғеҠЁеңЁз¬¬вјҖдёӘеҮҪж•°вҫҘжҺЁеҜје№¶ж·»еҠ дәҶвјҖдёӘ 'static зәҰжқҹпјҢжүҖд»Ҙе…¶е®һдёӨдёӘеҮҪж•°йғҪеҗ«жңү 'static зәҰжқҹгҖӮRust зј–иҜ‘еҷЁе®һйҷ…зңӢеҲ°зҡ„жҳҜиҝҷдёӘж ·вјҰзҡ„пјҡ

7) зј–иҜ‘еҷЁзҡ„жҠҘй”ҷдҝЎжҒҜдјҡе‘ҠиҜүжҲ‘жҖҺж ·дҝ®еӨҚжҲ‘зҡ„зЁӢеәҸ

  • Rust еҜ№ trait еҜ№иұЎзҡ„вҪЈе‘Ҫе‘ЁжңҹзңҒз•Ҙи§„еҲҷжҖ»жҳҜжӯЈзЎ®зҡ„

жҠҘй”ҷеҰӮдёӢ

еҘҪпјҢи®©жҲ‘д»¬жҢүз…§зј–иҜ‘еҷЁзҡ„жҸҗзӨәиҝӣвҫҸдҝ®еӨҚгҖӮиҝҷвҫҘжҲ‘д»¬е…ҲеҝҪз•ҘвјҖдёӘдәӢе®һпјҡиҝ”еӣһеҖјдёӯиЈ…з®ұзҡ„ trait еҜ№иұЎжңүвјҖдёӘвҫғеҠЁжҺЁеҜјзҡ„ 'static зәҰжқҹпјҢвҪҪзј–иҜ‘еҷЁжҳҜеҹәдәҺиҝҷдёӘжІЎжңүжҳҫејҸиҜҙжҳҺзҡ„дәӢе®һз»ҷеҮәзҡ„дҝ®еӨҚе»әи®®гҖӮ

зҺ°еңЁеҸҜд»Ҙзј–иҜ‘йҖҡиҝҮдәҶпјҢдҪҶиҝҷзңҹзҡ„жҳҜжҲ‘д»¬жғіиҰҒзҡ„еҗ—пјҹеҸҜиғҪжҳҜпјҢд№ҹеҸҜиғҪдёҚжҳҜпјҢзј–иҜ‘еҷЁе№¶жІЎжңүжҸҗеҲ°е…¶д»–дҝ®еӨҚвҪ…жЎҲпјҢдҪҶдёӢвҫҜиҝҷдёӘд№ҹжҳҜвјҖдёӘеҗҲйҖӮзҡ„дҝ®еӨҚвҪ…жЎҲгҖӮ

иҝҷдёӘеҮҪж•°жүҖиғҪжҺҘеҸ—зҡ„е®һйҷ…еҸӮж•°вҪҗеүҚвјҖдёӘеҮҪж•°еӨҡдәҶдёҚе°‘пјҒиҝҷдёӘеҮҪж•°жҳҜдёҚжҳҜжӣҙеҘҪпјҹзЎ®е®һпјҢдҪҶдёҚвјҖе®ҡеҝ…иҰҒпјҢиҝҷеҸ–еҶідәҺжҲ‘д»¬еҜ№зЁӢеәҸзҡ„иҰҒжұӮдёҺзәҰжқҹгҖӮдёҠвҫҜиҝҷдёӘдҫӢвјҰжңүзӮ№жҠҪиұЎпјҢжүҖд»Ҙи®©жҲ‘д»¬зңӢвјҖдёӘжӣҙз®ҖеҚ•жҳҺдәҶзҡ„дҫӢвјҰпјҡ

  • Rust еҜ№ trait еҜ№иұЎзҡ„вҪЈе‘Ҫе‘ЁжңҹзңҒз•Ҙи§„еҲҷе№¶дёҚдҝқиҜҒеңЁд»»дҪ•жғ…еҶөдёӢйғҪжӯЈзЎ®
  • еңЁзЁӢеәҸзҡ„иҜӯд№үвҪ…вҫҜпјҢRust е№¶дёҚвҪҗдҪ жҮӮ
  • Rust зј–иҜ‘й”ҷиҜҜзҡ„жҸҗзӨәдҝЎжҒҜжүҖжҸҗеҮәзҡ„дҝ®еӨҚвҪ…жЎҲе№¶дёҚвјҖе®ҡиғҪж»ЎвҫңдҪ еҜ№зЁӢеәҸзҡ„йңҖжұӮ

8) вҪЈе‘Ҫе‘ЁжңҹеҸҜд»ҘеңЁиҝҗвҫҸж—¶еҠЁжҖҒеҸҳв»“жҲ–еҸҳзҹӯ

й”ҷиҜҜзҡ„жҺЁи®ә

  • е®№еҷЁзұ»еҸҜд»ҘеңЁиҝҗвҫҸж—¶дәӨжҚўе…¶еҶ…йғЁзҡ„еј•вҪӨпјҢд»ҺвҪҪж”№еҸҳвҫғиә«зҡ„вҪЈе‘Ҫе‘Ёжңҹ
  • Rust еҖҹвҪӨжЈҖжҹҘеҷЁиғҪиҝӣвҫҸвҫјзә§зҡ„жҺ§еҲ¶жөҒеҲҶжһҗ

е°Ҫз®ЎдҪ еҸҜд»ҘеңЁеҮҪж•°е’ҢвҪ…жі•зҡ„еЈ°жҳҺвҫҘйҒҝе…ҚйҮҚеҖҹвҪӨпјҢдҪҶжҳҜз”ұдәҺ Rust дјҡвҫғеҠЁеҒҡйҡҗејҸйҮҚеҖҹвҪӨпјҢжүҖд»ҘеҫҲе®№жҳ“вҪҶж„ҸиҜҶең°йҒҮеҲ°иҝҷз§Қжғ…еҶө

дёҠвҫҜиҝҷж®өд»Јз Ғдјҡзј–иҜ‘еӨұиҙҘгҖӮиҝҷвҫҘ or_default() дјҡиҝ”еӣһвјҖдёӘ &mut PlayerпјҢдҪҶжҳҜз”ұдәҺжҲ‘д»¬ж·»еҠ дәҶвјҖдёӘжҳҫејҸзҡ„зұ»еһӢж ҮжіЁпјҢе®ғдјҡиў«йҡҗејҸйҮҚеҖҹвҪӨжҲҗ &PlayerгҖӮвҪҪдёәдәҶиҫҫжҲҗжҲ‘д»¬зңҹжӯЈзҡ„вҪ¬зҡ„пјҢжҲ‘д»¬дёҚеҫ—дёҚиҝҷж ·еҒҡпјҡ

йҡҫвҪӨпјҢвҪҪдё”еҫҲи ўпјҢдҪҶиҝҷжҳҜжҲ‘д»¬дёәдәҶеҶ…еӯҳе®үе…ЁиҝҷвјҖдҝЎжқЎжүҖеҒҡеҮәзҡ„зүәзүІгҖӮ

  • йҮҚеҖҹвҪӨвјҖдёӘзӢ¬еҚ еј•вҪӨе№¶дёҚдјҡз»“жқҹе…¶вҪЈе‘Ҫе‘ЁжңҹпјҢе“ӘжҖ•е®ғвҫғиә«е·Із»Ҹиў« drop жҺүдәҶ

10) еҜ№й—ӯеҢ…зҡ„вҪЈе‘Ҫе‘ЁжңҹзңҒз•Ҙи§„еҲҷе’ҢеҮҪж•°вјҖж ·

иҝҷжӣҙеғҸжҳҜ Rust зҡ„йҷ·йҳұвҪҪвҫ®иҜҜи§Је°Ҫз®Ўй—ӯеҢ…еҸҜд»Ҙиў«еҪ“дҪңжҳҜвјҖдёӘеҮҪж•°пјҢдҪҶжҳҜе№¶дёҚйҒөеҫӘе’ҢеҮҪж•°еҗҢж ·зҡ„вҪЈе‘Ҫе‘ЁжңҹзңҒз•Ҙи§„еҲҷгҖӮ

жҠҘй”ҷпјҡ

еҺ»жҺүиҜӯжі•зі–еҗҺпјҢжҲ‘д»¬еҫ—еҲ°зҡ„жҳҜпјҡ

еҮәзҺ°иҝҷз§Қе·®ејӮе№¶жІЎжңүд»Җд№ҲеҘҪеӨ„гҖӮеҸӘжҳҜеңЁй—ӯеҢ…жңҖеҲқзҡ„е®һзҺ°дёӯпјҢдҪҝвҪӨзҡ„зұ»еһӢжҺЁж–ӯиҜӯд№үдёҺеҮҪж•°дёҚеҗҢпјҢвҪҪзҺ°еңЁе°ҶвјҶиҖ…еҒҡвјҖдёӘз»ҹвјҖе°ҶжҳҜвјҖдёӘ breaking change, еӣ жӯӨзҺ°еңЁе·Із»ҸжІЎжі•ж”№дәҶгҖӮйӮЈд№ҲжҲ‘д»¬жҖҺд№ҲжҳҫејҸең°ж ҮжіЁвјҖдёӘй—ӯеҢ…зҡ„зұ»еһӢе‘ўпјҹжҲ‘д»¬жңүд»ҘдёӢвјҸз§ҚвҪ…жЎҲпјҡ

жҲ‘жғідҪ еә”иҜҘжіЁж„ҸеҲ°дәҶпјҢеңЁдёҠвҫҜзҡ„дҫӢвјҰдёӯпјҢеҰӮжһңеҜ№й—ӯеҢ…еә”вҪӨ trait зәҰжқҹпјҢй—ӯеҢ…дјҡе’ҢеҮҪж•°йҒөеҫӘеҗҢж ·зҡ„вҪЈе‘Ҫе‘ЁжңҹзңҒз•Ҙи§„еҲҷгҖӮиҝҷвҫҘжІЎжңүд»Җд№ҲзҺ°е®һзҡ„ж•ҷи®ӯжҲ–в»…и§ЈпјҢеҸӘжҳҜиҜҙжҳҺвјҖдёӢй—ӯеҢ…жҳҜиҝҷж ·зҡ„гҖӮ

11) 'static еј•вҪӨжҖ»иғҪиў«ејәеҲ¶иҪ¬жҚўдёә 'a еј•вҪӨ

жҲ‘д№ӢеүҚжңүиҝҮиҝҷж ·зҡ„д»Јз Ғпјҡ

дёӨиҖ…д№Ӣй—ҙжҳҜеҗҰжңүе®һйҷ…зҡ„е·®ејӮгҖӮжҲ‘вјҖејҖе§Ӣе№¶дёҚзЎ®е®ҡпјҢдҪҶвјҖз•Әз ”з©¶иҝҮеҗҺйҒ—жҶҫең°еҸ‘зҺ°пјҢжҳҜзҡ„пјҢиҝҷвјҶиҖ…зЎ®е®һжңүе·®ејӮгҖӮйҖҡеёёеңЁдҪҝвҪӨеҖјж—¶пјҢжҲ‘д»¬иғҪвҪӨ 'static еј•вҪӨзӣҙжҺҘд»ЈжӣҝвјҖдёӘ 'a еј•вҪӨпјҢеӣ дёә Rust дјҡвҫғеҠЁжҠҠ 'static еј•вҪӨејәеҲ¶иҪ¬жҚўдёә 'a еј•вҪӨгҖӮзӣҙи§үдёҠиҝҷеҫҲеҗҲзҗҶпјҢеӣ дёәеңЁвјҖдёӘеҜ№вҪЈе‘Ҫе‘ЁжңҹиҰҒжұӮвҪҗиҫғзҹӯзҡ„ең°вҪ…вҪӨвјҖдёӘвҪЈе‘Ҫе‘ЁжңҹвҪҗиҫғв»“зҡ„еј•вҪӨз»қдёҚдјҡеҜјиҮҙд»»дҪ•еҶ…еӯҳе®үе…Ёй—®йўҳгҖӮдёӢвҫҜзҡ„иҝҷж®өд»Јз ҒйҖҡиҝҮзј–иҜ‘пјҢе’Ңйў„жңҹвјҖиҮҙпјҡ

з„¶вҪҪеҪ“еј•вҪӨдҪңдёәеҮҪж•°зұ»еһӢзӯҫеҗҚзҡ„вјҖйғЁеҲҶж—¶пјҢејәеҲ¶зұ»еһӢиҪ¬жҚўе№¶дёҚвҪЈж•ҲгҖӮжүҖд»ҘдёӢвҫҜиҝҷж®өд»Јз ҒдёҚиғҪйҖҡиҝҮзј–иҜ‘пјҡ

жҠҘй”ҷеҰӮдёӢпјҡ

еҫҲйҡҫиҜҙиҝҷжҳҜдёҚжҳҜ Rust зҡ„вјҖдёӘйҷ·йҳұпјҢжҠҠ for<T> Fn() -> &'static T ејәеҲ¶иҪ¬жҚўдёә for<'a, T> Fn() -> &'a T е№¶дёҚжҳҜвјҖдёӘеғҸжҠҠ &'static str ејәеҲ¶иҪ¬жҚўдёә &'a str иҝҷж ·з®ҖеҚ•зӣҙвҪ©зҡ„жғ…еҶөгҖӮеүҚиҖ…жҳҜзұ»еһӢд№Ӣй—ҙзҡ„иҪ¬жҚўпјҢеҗҺиҖ…жҳҜеҖјд№Ӣй—ҙзҡ„иҪ¬жҚўгҖӮ

е…ій”®зӮ№еӣһйЎҫ

  • for <'aпјҢT> fnпјҲпјү->пјҶ'a T зӯҫеҗҚзҡ„еҮҪж•°вҪҗ for <T> fnпјҲпјү->пјҶ'static T зӯҫеҗҚзҡ„еҮҪж•°иҰҒжӣҙзҒөжҙ»пјҢе№¶дё”жіӣвҪӨдәҺжӣҙеӨҡеңәжҷҜ

3. жҖ»з»“

еӣ дёәйқҷжҖҒеҲҶжһҗжҠҖжңҜзҡ„еұҖйҷҗд»ҘеҸҠ Rust зҡ„дҝқе®ҲжҖ§пјҢеҸҜд»ҘзңӢеҲ° Rust еңЁеҶ…еӯҳе®үе…ЁеҒҡдәҶеҫҲеӨҡзңӢдјјеҫҲеӮ»зҡ„е®һзҺ°пјҲеҜ№дәҺ C++ вҫјвјҝжқҘиҜҙе®һйҷ…дёҠд№ҹеҫҲеӮ»зҡ„пјүгҖӮеҜ№дәҺжҒјзҫһжҲҗжҖ’зҡ„ Rust ж–°вјҝпјҲеҢ…жӢ¬дҪңиҖ…жң¬вјҲпјүпјҢеҸҜд»ҘдјҡвјҖжҖ’д№ӢдёӢе°ұ unsafe{} е°ұејҖе§ӢеғҸеҶҷ C вјҖж ·еҶҷ Rust дәҶпјҢдҪҶиҝҷжҳҫз„¶иҝқиғҢдәҶ Rust иҝҷв»”иҜӯвҫ”зҡ„еҲқиЎ·гҖӮеҗҢж—¶иҝҷвҫҘвҫҜжҸҗеҲ°зҡ„еҹәдәҺвҪЈе‘Ҫе‘ЁжңҹеҸҜиғҪеҸ‘вҪЈзҡ„еҶ…еӯҳе®үе…Ёй—®йўҳпјҢд№ҹиғҪз»ҷ modern C++ зҡ„зЁӢеәҸе‘ҳеёҰжқҘеҗҜеҸ‘пјҢе№¶дёҚжҳҜжҮӮзӮ№жҢҮй’Ҳе’ҢвҫҜеҗ‘еҜ№иұЎе°ұиғҪиҜҙвҫғвј°дјҡ C++ зҡ„пјҢвјҖдёӘиө„ж·ұзҡ„ C++ зЁӢеәҸе‘ҳиҰҒж—¶еҲ»иҖғиҷ‘еҶ…еӯҳе®үе…Ёй—®йўҳгҖӮsafe Rust зҡ„еҘҪеӨ„е°ұжҳҜпјҢйҖҡиҝҮзңӢдјјеҫҲеӮ»зҡ„вҪЈе‘Ҫе‘ЁжңҹжңәеҲ¶пјҢе°ҪеҸҜиғҪзҡ„еҺ»её®зЁӢеәҸе‘ҳи§„йҒҝеҶ…еӯҳе®үе…Ёй—®йўҳгҖӮ

  • T жҳҜ &T е’Ң &mut T зҡ„и¶…йӣҶ
  • &T е’Ң &mut T жҳҜдёҚзӣёдәӨзҡ„йӣҶеҗҲ
  • T: 'static еә”еҪ“и§Ҷдёә вҖң T ж»Ўвҫң 'static вҪЈе‘Ҫе‘ЁжңҹзәҰжқҹвҖқ
  • иӢҘ T: 'static еҲҷ T еҸҜд»ҘжҳҜвјҖдёӘжңү 'static вҪЈе‘Ҫе‘Ёжңҹзҡ„еј•вҪӨзұ»еһӢ жҲ– жҳҜвјҖдёӘжүҖжңүжқғзұ»еһӢ
  • еӣ дёә T: 'static еҢ…жӢ¬дәҶжүҖжңүжқғзұ»еһӢпјҢжүҖд»Ҙ T
    • еҸҜд»ҘеңЁиҝҗвҫҸж—¶еҠЁжҖҒеҲҶй…Қ
    • дёҚйңҖиҰҒеңЁж•ҙдёӘзЁӢеәҸиҝҗвҫҸжңҹй—ҙйғҪжңүж•Ҳ
    • еҸҜд»Ҙе®үе…ЁпјҢвҫғз”ұең°дҝ®ж”№
    • еҸҜд»ҘеңЁиҝҗвҫҸж—¶иў«еҠЁжҖҒзҡ„ drop

  • вјҸд№ҺжүҖжңүзҡ„ Rust д»Јз ҒйғҪжҳҜжіӣеһӢд»Јз ҒпјҢе№¶дё”еҲ°еӨ„йғҪеёҰжңүиў«зңҒз•ҘжҺүзҡ„жіӣеһӢвҪЈе‘Ҫе‘ЁжңҹжіЁи§Јe
  • Rust вҪЈе‘Ҫе‘ЁжңҹзңҒз•Ҙи§„еҲҷе№¶дёҚдҝқиҜҒеңЁд»»дҪ•жғ…еҶөдёӢйғҪжӯЈзЎ®
